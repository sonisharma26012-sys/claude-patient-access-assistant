# Claude-Powered Patient Access & No-Show Prevention Assistant

## Overview

This project analyzes healthcare appointment attendance patterns using over 110,000 real-world medical appointment records.

The platform identifies patient access barriers, missed appointment trends, operational bottlenecks, and high-risk populations while demonstrating how AI can transform healthcare analytics into actionable insights for non-technical stakeholders.

## Dataset

Medical Appointment No Shows Dataset

* 110,526 appointments
* 62,298 unique patients
* 81 neighborhoods
* Real healthcare appointment records
